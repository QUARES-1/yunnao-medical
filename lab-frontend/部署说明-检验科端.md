# 云脑诊疗平台检验科端部署说明

## 1. 本地开发

```bash
npm install
npm run dev
```

访问：`http://127.0.0.1:5176`

开发环境通过 `vite.config.ts` 将 `/api` 和 `/upload` 代理到后端 `http://localhost:8080`。

## 2. 生产打包

```bash
npm run build
```

打包完成后会生成 `dist/` 文件夹，生产部署只需要把 `dist/` 中的静态资源放到 Nginx 静态目录。

## 3. Nginx 双端分离部署

示例配置文件：`nginx-lab.conf`

推荐目录：

```text
/usr/share/nginx/html/cloud-brain-lab
```

部署步骤：

1. 将 `dist/` 内容复制到 `/usr/share/nginx/html/cloud-brain-lab/`。
2. 将 `nginx-lab.conf` 放到 Nginx 配置目录并启用。
3. 确保后端 Spring Boot 已启动在 `8080` 端口。
4. 重载 Nginx：`nginx -s reload`。

## 4. 拓展功能说明

- AI 流式响应：AI智能审核页面提供逐字审核输出，模拟真实 AI 审核生成过程，展示参考范围判断、逻辑校验、历史波动和处理建议。
- Pinia 状态管理：`src/stores/auth.ts` 管理检验师登录态，`src/stores/lab.ts` 管理待检队列、已完成报告、AI审核记录、人工复核列表、当前详情和流式输出状态。
- 双端分离部署：前端作为 Nginx 静态资源运行，接口统一通过 `/api` 反向代理到后端。
